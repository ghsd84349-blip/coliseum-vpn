# 🛡️ Coliseum VPN — Private Node Ecosystem

Мобильное приложение для частного VPN-сервиса, разработанное специально для портфолио в стиле киберпанк-клуба «Колизей Арена».

## 🚀 Как запустить интерактивное демо в 1 клик:
Нажмите на кнопку ниже, чтобы открыть и пощелкать приложение прямо в браузере (вводить можно любые логины/пароли):

[![Запустить во Flutter DartPad](https://img.shields.io/badge/DartPad-Run_Interactive_Demo-blue?style=for-the-badge&logo=flutter)](https://dartpad.dev/?null_safety=true&id=7e419b59-8913-4d7d-9c02-3d460162ca17)

## 🛠️ Технологии:
* **Frontend:** Flutter (Dart)
* **Архитектура:** Чистый UI, Mock Mode (симуляция трафика, графиков скорости и таймера сессии для презентаций).
* **Target Network:** WireGuard (в планах бэкенд на Python FastAPI).
